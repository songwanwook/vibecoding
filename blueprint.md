
# Lottery Number Generator

## Overview

This project is a simple, visually appealing web application to generate random lottery numbers. It's built with modern HTML, CSS, and JavaScript, following best practices for a clean and maintainable codebase.

## Current Plan

### Goal: Create a Lottery Number Generation Site

1.  **Structure (`index.html`):**
    *   Create a container for the lottery machine.
    *   Add a title: "Lotto Number Generator".
    *   Add a display area for the generated numbers.
    *   Add a "Generate Numbers" button.

2.  **Styling (`style.css`):**
    *   Apply a modern and clean design.
    *   Use a visually appealing color palette and fonts.
    *   Style the number display to be clear and engaging.
    *   Add hover and active states to the button for better user feedback.
    *   Incorporate a subtle background texture and drop shadows for depth.

3.  **Functionality (`main.js`):**
    *   Implement a function to generate 6 unique random numbers from 1 to 45.
    *   Attach an event listener to the "Generate Numbers" button.
    *   When the button is clicked, display the new numbers with a simple animation.
    *   Color-code the numbers for better visual distinction.
